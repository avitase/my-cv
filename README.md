# My CV

This repository hosts my CV in a deliberately plain Markdown source format (ATS-friendly, easy to diff).

## Downloads

Ready-to-use artifacts are attached to the [**Releases**](https://github.com/avitase/my-cv/releases) page of this repository:
- [MD](https://github.com/avitase/my-cv/releases/latest/download/cv_nis_meinert.md)
- [PDF](https://github.com/avitase/my-cv/releases/latest/download/cv_nis_meinert.pdf)
- [DOCX](https://github.com/avitase/my-cv/releases/latest/download/cv_nis_meinert.docx)
- [HTML](https://github.com/avitase/my-cv/releases/latest/download/cv_nis_meinert.html)
- [TXT](https://github.com/avitase/my-cv/releases/latest/download/cv_nis_meinert.txt)

## Build locally

```sh
make
```

Outputs are written to `build/` (requires [pandoc](https://pandoc.org/)).
